# tslint-enum-value-name-rule

It is a TSLint rule to enforce PascalCasing in enum names.

## Installation

```bash
npm i -D tslint-enum-value-name-rule
```

In `tslint.json` config

```json
{
  "extends": [
    "tslint-enum-value-name-rule"
  ],
  "rules": {
    "enum-value-name": true
  }
}
```
